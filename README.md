# 123wild - A wild life challenge

## what is 123 wild?
### breif
123wild is a wildlife photography sharing platform, where upon uploading 123 pictures of the named animal or birds from the list you'll recive a wildlife certification
### storage
123wild uses firebase free cloudstorage to store all the images uploaded.
### verification
123wild uses deeplearning to check for the accuracy and authenticity of the image uploaded, it uses a two step supervised model to analyse the image. [to be implemented]
